Function Reference — OpenCV & Python Demo

This post documents the core functions provided in the combined demo script and points you to the full source on GitHub.

Image utilities

load_image(path) — Load an image using OpenCV and return the BGR image or None if loading failed.

show_image(winname, img, wait=True) — Display an image in a named window. If wait is True the function waits for a keypress and closes the window.

print_image_info(img, name='image') — Print image metadata: shape, size, and dtype.

access_pixel(img, x, y) — Safely read the BGR value of a pixel and print its blue channel. Returns the pixel value.

modify_pixels(img, modifications) — Modify pixels in-place. Pass a dictionary like {(x,y): (B,G,R)} and the function will update those coordinates.

Video utilities

capture_video(device=0) — Open the system camera (default device 0) and stream frames to a window named 'frame'. Press ESC to stop capture and close the window.

Image analysis

check_image_load_and_show(path) — Convenience wrapper that attempts to load an image and shows it if successful, otherwise logs an error.

classify_brightness(path) — Loads an image, converts to grayscale for display, computes an average brightness and classifies it as dark, medium, or bright.

Small Python demos

list_demo() — Runs a set of list operations demonstrating indexing, slicing, append/insert/remove, and iteration.

functions_demo() — Demonstrates function types: normal functions, default args, *args, **kwargs, and a lambda example.

while_loop_demo() and for_loop_demo() — Simple examples of loop usage and control flow.

CLI and orchestration

The script provides a main() and command-line arguments so you can run individual demos or --all to run everything. The CLI parsing is implemented with argparse.
