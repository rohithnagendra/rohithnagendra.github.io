OpenCV & Matplotlib Master — Functions Explained

A friendly walkthrough of each function included in the combined Python script. Run the script locally and use the CLI examples below.

Quick overview

This page documents a single-file Python utility that groups many small OpenCV and Matplotlib demos into functions with a CLI wrapper. It is designed for learning, experimentation, and as a reference when you want to run any demo quickly.

Primary file: opencv_matplotlib_master.py

play_video(video_path: str = "vtest.avi", delay_ms: int = 10)

Opens a video file or webcam stream and plays it in an OpenCV window. Press ESC to exit playback.

What it does

  • Accepts a filename or numeric webcam id (e.g. "0").
  • Reads frames in a loop and shows them with cv.imshow.
  • Releases the capture and destroys windows on exit.
CLI example: 
python opencv_matplotlib_master.py --video vtest.avi
# or to use webcam
python opencv_matplotlib_master.py --video 0

show_image_rgb_plot(img_path: str = "lena.jpg")

Loads an image with OpenCV and displays it both with OpenCV (BGR) and Matplotlib (converted to RGB).

Why it matters

OpenCV uses BGR ordering while Matplotlib expects RGB. This helper demonstrates correct conversion to avoid color shifts when plotting.

CLI example: 
python opencv_matplotlib_master.py --hsv lena.jpg
# (this will show BGR via OpenCV and RGB via Matplotlib)

roi_copy_paste_example(img_path: str = "hazard10.jpg")

Demonstrates extracting a region of interest (ROI), copying it, and pasting it elsewhere in the image. Also shows how to paint a rectangular region with a fixed BGR color.

Notes

  • Coordinates were taken from the original example. For different images you will likely want to update the slice ranges.
  • The function uses .copy() when extracting ROI to avoid accidental references into the same memory.
python opencv_matplotlib_master.py --roi hazard10.jpg

show_hsv_conversion_and_plot(img_path: str = "lena.jpg")

Converts an image to HSV and displays both the original and HSV-converted image — HSV converted to RGB for proper Matplotlib visualization.

Teaching points

  • HSV separates chromatic information (H,S) from intensity (V), which is useful for color-based segmentation.
  • Always convert HSV back to RGB before plotting with Matplotlib.
python opencv_matplotlib_master.py --hsv lena.jpg

split_and_show_channels(img_path: str = "lena.jpg")

Splits the image into B, G, R channels and also into H, S, V channels for demonstration. Shows each channel using OpenCV windows.

Important

Channel images are single-channel gray images. You can merge them back with cv.merge.

python opencv_matplotlib_master.py --split lena.jpg

show_image_info_and_conversions(img_path: str = "opencv-logo.png")

Prints shape & size metadata and shows conversions: grayscale, RGB, HSV. Useful to inspect array dimensions and channel counts.

python opencv_matplotlib_master.py --info opencv-logo.png

draw_shapes_example(save_to: str = None)

Creates a blank canvas and draws common primitives: line, rectangle, circle, polygon, and text. Optionally saves the resulting image.

python opencv_matplotlib_master.py --draw
# or save to a file
python opencv_matplotlib_master.py --draw shapes.png

Matplotlib plotting helpers

Small functions that produce example plots: plot_line_example, plot_bar_example, plot_sine_example, plot_scatter_example.

python opencv_matplotlib_master.py --plot line
python opencv_matplotlib_master.py --plot bar
python opencv_matplotlib_master.py --plot sine
python opencv_matplotlib_master.py --plot scatter

--all (run demos sequentially)

A convenience flag that attempts to run the demos one after another. Useful for a quick tour — note it will open windows and show plots in sequence and may require multiple key presses to proceed.

python opencv_matplotlib_master.py --all
